# SimpleTooltipPlus

**SimpleTooltipPlus** is a lightweight World of Warcraft addon designed to enhance the default unit tooltip with essential information. It provides a quick look at player item levels and unit targets.

## Features

- **Accurate Item Level:** Displays the average item level of players.
- **Target Display:** Shows the "Target of Target" for both players and NPCs.
  - **Class Colors:** Player targets are colored by their class.
  - **Contextual Styling:** NPC targets are displayed in a neutral grey for better visual hierarchy.
- **Mounts:** Shows the current players mount and your collection status (Collected / Not collected).

## Installation

1. Download the latest release.
2. Extract the `SimpleTooltipPlus` folder.
3. Move the folder to your World of Warcraft directory:
   `_retail_/Interface/AddOns/`
4. Restart the game or type `/reload` in-game.
